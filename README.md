# 淘宝首页

## Live Demo
[https://imtypist.github.io/TaoBao/](https://imtypist.github.io/TaoBao/)

* 因为webp图片格式的原因，请在chrome浏览器下打开

## 环境准备

* Sublime text
* git

## 操作

* 先把代码git到本地

```bash
git clone https://github.com/imtypist/TaoBao.git
```
* 修改远程仓库地址

```bash
git remote rm origin
git remote add origin {your repository url}
```

## 介绍

* 采用DIV+CSS布局模式模拟淘宝页面

* 对于训练前端入门的布局能力十分有效

## 注意事项

* class/id的命名要规范

* 采用外链样式表

* 整体考虑布局，先大后小
